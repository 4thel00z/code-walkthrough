# code-walkthrough

## Motivation

In the age of the AI craze it becomes increasingly important to create an understadning of codebases - that may or may not be produced by AI agents,
in a rapid, deep, extensive and visual manner.

For this purpose, we built this skill called "code-walkthrough", to be able to generate step-by-step code walkthroughs, which are enhanced by graphs/diagrams,
and other visual means, to aid the understanding of a code base.
